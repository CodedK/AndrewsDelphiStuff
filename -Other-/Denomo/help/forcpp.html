From verion 2.1, Denomo can not only be used for Delphi applications, but also used for C++ programs.

Now it has been test that Denomo can be used in Borland C++ Builder 5.5, 6.0 and Microsoft Visual C++ 6.0, 2008 Express Edition. Higher versions are supposed to support, but not verified yet.
Other C++ compilers are not tested.

To use the C++ version, you need to download Denomo for C++.
